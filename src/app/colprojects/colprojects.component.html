Mini-Project

Currency Recognition For Visually Impaired People Indian

The Android Studio app assists visually impaired individuals by recognizing Indian currency through advanced image processing. It employs pattern recognition to determine denominations and provides real-time voice or tactile feedback, enhancing independence in daily financial transactions. This innovative tool combines technology and accessibility to empower users in confidently handling currency-related tasks.

Major-Project

Cognitive radio network for minimizing network traffic

An Adaptive Radio Network mitigates network congestion. Cognitive Radio Networks (CRN) intelligently manage spectrum utilization, employing advanced machine learning and spectrum sensing to detect and allocate underutilized frequency bands dynamically. CRNs adapt to real-time network conditions, promoting efficient coexistence and reducing congestion, thereby improving overall wireless network performance and user experiences.
